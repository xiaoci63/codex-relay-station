## Codex Relay Station - CC Switch 接入指南

### 你的配置信息

| 配置项 | 值 |
|--------|-----|
| API 地址 | `http://108.160.141.175:3000/v1` |
| API Key | `sk-xxxxx`（管理员分配给你的密钥） |
| 推荐模型 | `gpt-5.5` |

---

### 方法一：CC Switch 一键导入（推荐）

如果你的管理员给你发了一个 `ccswitch://` 开头的链接，直接点击即可自动导入配置，无需手动设置。

> 管理员可在 New API 管理面板 → 令牌管理 → 点击令牌的 CC Switch 图标 生成导入链接。

---

### 方法二：手动配置

#### 第一步：打开 CC Switch

启动 CC Switch 应用。如果还没安装，前往 GitHub 下载：
https://github.com/farion1231/cc-switch

#### 第二步：选择目标应用

在顶部应用切换器中，选择你要配置的 CLI 工具（Claude Code / Codex / Gemini CLI 等）。

#### 第三步：添加供应商

1. 点击 **"+"** 按钮添加新供应商
2. 在「预设」下拉框中选择 **"手动配置"**（不选预设）

#### 第四步：填写配置

| 字段 | 填写内容 |
|------|---------|
| **名称** | `Codex Relay`（或任意你喜欢的名字） |
| **API Key** | 管理员给你的 `sk-xxxxx` 密钥 |
| **Base URL** | `http://108.160.141.175:3000/v1` |
| **模型** | `gpt-5.5` |
| **API 格式** | `OpenAI Chat Completions` |

#### 第五步：启用并验证

1. 在供应商列表中，启用刚才添加的 `Codex Relay`
2. 打开终端，运行你的 CLI 工具
3. 如果看到正常响应，说明配置成功

---

### 可用模型列表

除了 `gpt-5.5`，还支持以下模型：

| 模型 | 适用场景 |
|------|---------|
| `gpt-5.5` | 旗舰模型，复杂任务首选 |
| `gpt-4o` | 通用对话和编程，速度快 |
| `gpt-4o-mini` | 轻量任务，最省额度 |
| `o1` | 深度推理 |
| `o1-mini` | 快速推理 |
| `o3` | 高级推理 |
| `o3-mini` | 轻量推理 |
| `o4-mini` | 最新推理模型 |

> 建议：日常编程用 `gpt-4o` 或 `gpt-4o-mini`，遇到复杂问题再切 `gpt-5.5`，避免频繁触发速率限制。

### 常见问题

**Q: 连接超时或报错？**
检查网络是否能访问 `108.160.141.175:3000`，可能需要梯子或代理。

**Q: 提示 "额度不足"？**
联系管理员充值额度。

**Q: 提示 "速率限制"？**
ChatGPT Plus 有消息频率限制，等几分钟再试。

**Q: 模型不可用？**
确认你填写的模型名称正确，注意大小写（全部小写）。
